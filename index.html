Algorithm walkthrough (before implementation)

  1. Normalize each horse on core factors (speed, form, class, pace fit, distance fit, connections, consistency, volatility, late kick, trend).
  2. Build a base score from weighted performance factors, then subtract a risk penalty for volatility.
  3. Convert base scores into model win probabilities and compare them to implied probabilities from projected odds.
  4. Pull BRISNET signals (Spot Plays and Optix mentions) and translate them into a confidence factor that can shift horse rank.
  5. Add a value adjustment so overpriced horses can rise and overbet horses can fall.
  6. Rank the field and auto-build five tickets, plus an undercover winner outside top-tier bets.
  7. Generate a mirrored opposition slip so you can bet against the model using an explicit counter-position.
